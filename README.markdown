# STFU (formerly STFU and Code)
## Easily build C# without any editor what-so-ever
0. Download stfu.exe
1. stfu install (remove the downloaded exe)
2. stfu new your_project_name
3. stfu build

STFU! You're done.

## Built using STFU
Written in C#, for C#.
